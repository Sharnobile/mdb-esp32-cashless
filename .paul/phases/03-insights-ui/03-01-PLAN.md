## Goal
Create a `useInsights` composable and integrate an "AI Insights" button on the machine detail page (`/machines/[id]`) that calls the `machine-insights` edge function and displays structured recommendations in a Sheet overlay.

- Updated i18n files — EN + DE translation keys

<task type="auto">
  <name>Task 1: Create useInsights composable</name>
  <files>management-frontend/app/composables/useInsights.ts</files>
  <action>
    Create a composable following the `useMdbLog.ts` pattern:

    **Exported types:**
    ```typescript
    export interface InsightRecommendation {
      type: 'product_swap' | 'capacity_increase' | 'remove_slot' | 'refill_optimization' | 'conversion_alert' | 'general'
      priority: 'high' | 'medium' | 'low'
      title: string
      detail: string
      item_number: number | null
    }

    export interface InsightsResponse {
      generated_at: string
      period_days: number
      machine: { id: string; name: string }
      recommendations: InsightRecommendation[]
      summary: string
    }
    ```

    **Composable function `useInsights()`:**
    - State: `data` (ref<InsightsResponse | null>), `loading` (ref<boolean>), `error` (ref<string>)
    - Function `fetchInsights(machineId: string, days?: number)`:
      - Sets `loading = true`, clears `error`
      - Calls `useSupabaseClient().functions.invoke('machine-insights', { body: { machine_id: machineId, days: days ?? 30 } })`
      - On success: sets `data` to response, clears `error`
      - On error: sets `error` to the error message (parse the response body for `{ error: "..." }`)
      - Sets `loading = false` in finally block
    - Return: `{ data, loading, error, fetchInsights }`

    **Export pure helpers:**
    - `recommendationIcon(type: string): string` — returns a Tabler icon name for each recommendation type
    - `priorityVariant(priority: string): 'default' | 'secondary' | 'destructive'` — maps priority to Badge variant (high=destructive, medium=default, low=secondary)
  </action>

    File exists: `ls management-frontend/app/composables/useInsights.ts`

    Contains types: `grep "InsightRecommendation" management-frontend/app/composables/useInsights.ts`
  </verify>
  <done>AC-2, AC-3 satisfied: composable handles fetch, loading, error, and data states</done>
