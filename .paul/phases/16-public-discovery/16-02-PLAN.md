<objective>
## Goal
Build the global public machine discovery page at `/m/` — an interactive Leaflet + OpenStreetMap view of all vending machines with `public_listing = true` from all operators. Each marker links to its machine detail page. Accessible fallback list below the map.

## Purpose
Customers can discover nearby vending machines without knowing specific URLs. They can find the closest one with their favorite product, see online/offline status at a glance, and navigate directly to buy via the detail page.

## Output
- Edge function `public-machines-list` returning sanitized machine list
- Public Nuxt page `/m/index.vue` with Leaflet map + fallback list
- OpenStreetMap tile layer (no API key, DSGVO-friendly)
- i18n translations (en/de)
</objective>

## Architecture Notes
- **Leaflet via CDN** (`unpkg.com/leaflet@1.9.4`): no npm install needed, smaller bundle for public page, easy to load dynamically in client-only context
- **OpenStreetMap tiles**: `https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png` — free, no API key, DSGVO-friendly, requires attribution
- **Client-only map**: Leaflet needs `window` and DOM, mount in `onMounted`
- **Fit bounds**: map auto-centers/zooms to show all markers
- **List fallback**: accessibility + machines without lat/lon coordinates

    **Dynamic Leaflet loading** (CDN):
    ```typescript
    async function loadLeaflet(): Promise<any> {
      if ((window as any).L) return (window as any).L
      // Load CSS
      const link = document.createElement('link')
      link.rel = 'stylesheet'
      link.href = 'https://unpkg.com/leaflet@1.9.4/dist/leaflet.css'
      link.integrity = 'sha256-p4NxAoJBhIIN+hmNHrzRCf9tD/miZyoHS5obTRR9BMY='
      link.crossOrigin = ''
      document.head.appendChild(link)
      // Load JS
      return new Promise((resolve, reject) => {
        const script = document.createElement('script')
        script.src = 'https://unpkg.com/leaflet@1.9.4/dist/leaflet.js'
        script.integrity = 'sha256-20nQCchB9co0qIjJZRGuk2/Z9VM+kNiyxNV1lvTlZBo='
        script.crossOrigin = ''
        script.onload = () => resolve((window as any).L)
        script.onerror = reject
        document.head.appendChild(script)
      })
    }
    ```

    **Map initialization** (onMounted, after data loaded):
    ```typescript
    const mapContainer = ref<HTMLElement | null>(null)
    let mapInstance: any = null

    async function initMap() {
      if (!mapContainer.value || !data.value?.machines) return
      const L = await loadLeaflet()
      if (mapInstance) return  // already initialized

      const withCoords = data.value.machines.filter((m: any) => m.location_lat && m.location_lon)
      if (withCoords.length === 0) return

      mapInstance = L.map(mapContainer.value).setView([51.1657, 10.4515], 6) // Germany center fallback

      L.tileLayer('https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png', {
        attribution: '© <a href="https://www.openstreetmap.org/copyright">OpenStreetMap</a>',
        maxZoom: 18,
      }).addTo(mapInstance)

      // Add markers
      const bounds: any[] = []
      for (const machine of withCoords) {
        const isOnline = machine.status === 'online'
        const icon = L.divIcon({
          className: 'custom-marker',
          html: `<div class="size-6 rounded-full border-2 border-white shadow-lg ${isOnline ? 'bg-emerald-500' : 'bg-gray-500'}"></div>`,
          iconSize: [24, 24],
          iconAnchor: [12, 12],
        })
        const marker = L.marker([machine.location_lat, machine.location_lon], { icon }).addTo(mapInstance)
        const popupHtml = `
          <div class="min-w-[160px]">
            <div class="font-semibold mb-1">${machine.name}</div>
            ${machine.company_name ? `<div class="text-xs text-muted-foreground mb-2">${machine.company_name}</div>` : ''}
            <div class="mb-2"><span class="inline-block rounded-full px-2 py-0.5 text-xs ${isOnline ? 'bg-emerald-500/15 text-emerald-600' : 'bg-gray-500/15 text-gray-600'}">${isOnline ? 'Online' : 'Offline'}</span></div>
            <a href="/m/${machine.id}" class="text-sm font-medium text-primary hover:underline">${t('publicMap.viewDetails')} →</a>
          </div>
        `
        marker.bindPopup(popupHtml)
        bounds.push([machine.location_lat, machine.location_lon])
      }

      // Fit bounds if multiple markers
      if (bounds.length > 1) {
        mapInstance.fitBounds(bounds, { padding: [40, 40] })
      } else if (bounds.length === 1) {
        mapInstance.setView(bounds[0], 15)
      }
    }

    onMounted(() => { initMap() })
    // Re-init if data changes (e.g., after refresh)
    watch(data, () => { if (data.value) initMap() })
    ```
