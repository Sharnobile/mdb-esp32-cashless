<objective>
## Goal
Add a "Public Visibility" section to the Machine Detail Page (`/machines/[id]`) — admin-only UI to toggle `public_listing`, copy the public URL `/m/[machine_id]`, and download a QR code pointing to the public page. Operators can now control which machines appear in discovery views and easily print QR codes for the physical machines.

## Purpose
Closes the feature loop for Phase 16: everything built in Plans 01-03 (stable URLs, global map, operator page) is now manageable from the operator dashboard. No more needing to run SQL or build QR codes externally.

## Output
- Public Visibility card on Machine Detail Page with toggle + URL + QR download
- Admin-only (uses existing `isAdmin` check)
- i18n (en/de)
</objective>

- `useRuntimeConfig().public.supabase.url` provides the base URL

## Architecture Notes
- The public URL is `{appBaseUrl}/m/{machine_id}` — NOT the Supabase URL. We need the frontend domain.
- For the QR code to work from a phone scan, the URL must be reachable from phones (LAN IP or public domain)
- Best approach: use `window.location.origin` at runtime (client-side only) — this guarantees the user gets whatever domain they're currently on
- Server-side rendering: show the URL as a placeholder, populate on mount
- QR code generated only when toggle is ON (saves CPU when feature not used)

    ```typescript
    const publicListingLoading = ref(false)
    const publicListingError = ref('')
    const publicUrlCopied = ref(false)
    const publicQrDataUrl = ref<string>('')
    const publicPageOrigin = ref<string>('') // Populated on mount (client-only)

    const publicUrl = computed(() => {
      if (!machine.value?.id || !publicPageOrigin.value) return ''
      return `${publicPageOrigin.value}/m/${machine.value.id}`
    })

    async function togglePublicListing() {
      if (!machine.value) return
      const newValue = !machine.value.public_listing
      publicListingLoading.value = true
      publicListingError.value = ''
      // Optimistic update
      const prevValue = machine.value.public_listing
      machine.value.public_listing = newValue
      try {
        const { error } = await supabase
          .from('vendingMachine')
          .update({ public_listing: newValue })
          .eq('id', machine.value.id)
        if (error) throw error
        // Regenerate QR code if now enabled
        if (newValue) {
          await generatePublicQr()
        } else {
          publicQrDataUrl.value = ''
        }
      } catch (err) {
        // Revert on error
        machine.value.public_listing = prevValue
        publicListingError.value = err instanceof Error ? err.message : 'Failed to update'
      } finally {
        publicListingLoading.value = false
      }
    }

    async function generatePublicQr() {
      if (!publicUrl.value) return
      try {
        publicQrDataUrl.value = await QRCode.toDataURL(publicUrl.value, {
          width: 256,
          margin: 2,
          errorCorrectionLevel: 'M',
        })
      } catch (err) {
        console.error('QR code generation failed:', err)
      }
    }

    async function copyPublicUrl() {
      if (!publicUrl.value) return
      try {
        await navigator.clipboard.writeText(publicUrl.value)
        publicUrlCopied.value = true
        setTimeout(() => { publicUrlCopied.value = false }, 2000)
      } catch (err) {
        console.error('Copy failed:', err)
      }
    }

    function downloadQrCode() {
      if (!publicQrDataUrl.value || !machine.value?.id) return
      const link = document.createElement('a')
      link.href = publicQrDataUrl.value
      link.download = `machine-${machine.value.id.substring(0, 8)}.png`
      document.body.appendChild(link)
      link.click()
      document.body.removeChild(link)
    }

    // Populate origin + generate QR on mount
    onMounted(() => {
      publicPageOrigin.value = window.location.origin
      if (machine.value?.public_listing) {
        generatePublicQr()
      }
    })

    // Regenerate QR when machine ID becomes available (after fetch)
    watch(() => machine.value?.id, (newId) => {
      if (newId && machine.value?.public_listing && publicPageOrigin.value) {
        generatePublicQr()
      }
    })
    ```
