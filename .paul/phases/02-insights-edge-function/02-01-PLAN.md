<objective>
## Goal
Create the `machine-insights` Supabase edge function that accepts a `machine_id`, calls the Phase 01 RPC, sends a structured prompt to the Anthropic Claude API, and returns JSON recommendations. Also register `ANTHROPIC_API_KEY` in all required env config files.

## Purpose
This is the AI brain of the feature — it connects the pre-computed KPIs (Phase 01) to an LLM and converts raw numbers into actionable operator recommendations: which products to swap, which slots need more capacity, when to refill, and whether conversion rate is a concern.

  <name>Task 1: Create machine-insights edge function</name>
  <files>Docker/supabase/functions/machine-insights/index.ts, Docker/supabase/functions/machine-insights/deno.json</files>
  <action>
    **deno.json** — minimal import map:
    ```json
    {
      "imports": {
        "@supabase/supabase-js": "https://esm.sh/@supabase/supabase-js@2",
        "@anthropic-ai/sdk": "npm:@anthropic-ai/sdk"
      }
    }
    ```

    **index.ts** — implement the function as follows:

    1. **Top-level imports:**
    ```typescript
    import { createClient } from '@supabase/supabase-js'
    import Anthropic from '@anthropic-ai/sdk'
    ```

    2. **Auth helper** — reuse the same dual-auth pattern from `send-credit/index.ts`:
       - Check `X-API-Key` header first: hash with SHA-256, look up `api_keys` table via service role, extract `company_id`
       - Else check `Authorization: Bearer <jwt>`: call `adminClient.auth.getUser(token)`, then query `organization_members` where `user_id = user.id` to get `company_id`
       - If neither: return 401 `{ error: 'Unauthorized' }`

    3. **Parse request body:**
    ```typescript
    const { machine_id, days = 30 } = await req.json()
    if (!machine_id) return 400 { error: 'machine_id required' }
    ```

    4. **Guard: check ANTHROPIC_API_KEY early:**
    ```typescript
    const anthropicKey = Deno.env.get('ANTHROPIC_API_KEY')
    if (!anthropicKey) return 500 { error: 'ANTHROPIC_API_KEY not configured' }
    ```

    5. **Call RPC** via service role client:
    ```typescript
    const { data: kpis, error: rpcError } = await adminClient
      .rpc('get_machine_insights_kpis', {
        p_machine_id: machine_id,
        p_company_id: companyId,
        p_days: days
      })
    if (rpcError) return 500 { error: rpcError.message }
    if (!kpis) return 404 { error: 'Machine not found' }
    ```

    6. **Build Claude prompt** — construct a compact, factual prompt:
    ```typescript
    const prompt = `You are an AI assistant helping vending machine operators optimize their machines.

    Analyze the following ${days}-day performance data for machine "${kpis.machine.name}" and provide actionable recommendations.

    MACHINE DATA:
    ${JSON.stringify(kpis, null, 2)}

    KPI DEFINITIONS:
    - sell_through_pct: % of theoretical weekly capacity sold. <30% = underperforming, >80% = strong
    - is_dead_stock: true if 0 sales in the period — consider replacing this product
    - days_until_empty: estimated days until this tray runs out at current rate
    - conversion_rate: sales / foot traffic. <0.05 = very low (pricing or product issue)
    - avg_daily_units: average units sold per day

    Return a JSON object with exactly this structure (no markdown, raw JSON only):
    {
      "recommendations": [
        {
          "type": "product_swap" | "capacity_increase" | "remove_slot" | "refill_optimization" | "conversion_alert" | "general",
          "priority": "high" | "medium" | "low",
          "title": "Short actionable title (max 60 chars)",
          "detail": "Specific detail with numbers (1-2 sentences)",
          "item_number": <integer or null>
        }
      ],
      "summary": "One paragraph narrative summary of overall machine performance and top 2-3 actions to take."
    }

    Rules:
    - Only recommend product_swap for slots where is_dead_stock=true or sell_through_pct < 20
    - Recommend capacity_increase for slots where days_until_empty < 5 and sell_through_pct > 70
    - Flag conversion_alert if conversion_rate < 0.03
    - Recommend refill_optimization if any tray has days_until_empty < 3
    - Maximum 6 recommendations. Prioritize high-impact ones.
    - If data is missing or machine has no sales, say so clearly in summary.`
    ```

    7. **Call Anthropic API:**
    ```typescript
    const anthropic = new Anthropic({ apiKey: anthropicKey })
    const message = await anthropic.messages.create({
      model: 'claude-haiku-4-5',
      max_tokens: 1024,
      messages: [{ role: 'user', content: prompt }]
    })
    const rawText = message.content[0].type === 'text' ? message.content[0].text : ''
    ```

    8. **Parse response** — Claude should return raw JSON. Parse with try/catch:
    ```typescript
    let parsed: { recommendations: unknown[]; summary: string }
    try {
      parsed = JSON.parse(rawText)
    } catch {
      // Fallback: extract JSON block from text if Claude wrapped it
      const match = rawText.match(/\{[\s\S]*\}/)
      parsed = match ? JSON.parse(match[0]) : { recommendations: [], summary: rawText }
    }
    ```

    9. **Return final response:**
    ```typescript
    return new Response(JSON.stringify({
      generated_at: new Date().toISOString(),
      period_days: days,
      machine: kpis.machine,
      recommendations: parsed.recommendations ?? [],
      summary: parsed.summary ?? ''
    }), { status: 200, headers: { 'Content-Type': 'application/json' } })
    ```

    10. **Wrap all in try/catch** returning 500 with error message on unexpected failures.

    IMPORTANT:
    - Use `Deno.serve(async (req) => { ... })` as the entry point (same as all other functions)
    - The `adminClient` uses `SUPABASE_URL` and `SUPABASE_SERVICE_ROLE_KEY` from env (auto-injected by Supabase runtime)
    - Do NOT use `verify_jwt = true` — auth is handled inside the function (matches all other functions)
    - Model name: `claude-haiku-4-5` (fast, cheap, suitable for structured output tasks)
    - Avoid using `anthropic.beta` or streaming — plain `messages.create` is sufficient
    - Log errors to console for debugging: `console.error('[machine-insights]', error)`
  </action>
