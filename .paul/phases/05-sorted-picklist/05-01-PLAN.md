<objective>
## Goal
Sort the refill wizard packing step's pick lists by warehouse product position and add a combined picking mode so operators can walk through the warehouse linearly instead of zigzagging.

## Purpose
When an operator prepares products for refill, the current pick list is sorted by deficit (largest first). This means the operator jumps back and forth in the warehouse. By sorting items according to `warehouse_product_positions.sort_order`, the operator walks from start to end of the warehouse exactly once. The combined mode merges all machines into one list for a single pass.

## Output
- `useRefillWizard` extended with position-based sorting and picking mode
- Refill packing UI updated with mode toggle and sorted items
- Combined picking view showing all products across all machines in warehouse order
</objective>
